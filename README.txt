--- INSTALLATION ---

Clone the RICEDDARIO package:

    $ git clone https://github.com/dcorrada/RICEDDARIO.git

Then, launch the installer script to update your bashrc file:

    $ cd RICEDDARIO;
    $ ./INSTALL.pl
